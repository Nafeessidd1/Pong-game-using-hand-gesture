# Pong-game-using-hand-gesture


This project is a simple implementation of the classic Pong game controlled by hand gestures. It utilizes OpenCV and the CVZone library in Python to detect and track hand movements in real-time, allowing players to interact with the game using hand gestures.


Features
Real-time hand gesture detection and tracking using OpenCV and CVZone.
Control the paddle movement using hand gestures (up and down).
Simple and intuitive user interface.
Single-player mode against an AI opponent.
Adjustable difficulty levels.
Installation
Clone this repository to your local machine:

git clone https://github.com/Nafeessidd1/Pong-game-using-hand-gesture.git

Navigate to the project directory:

cd Pong-game-using-hand-gesture

Install the required dependencies:

pip install -r requirements.txt
Usage
Run the main script to start the game:

python pong_game.py
Follow the on-screen instructions to control the paddle using hand gestures.
Enjoy playing the Pong game!

Dependencies

Python 3.x

OpenCV

CVZone

NumPy
